A Estrutura básica do HTML

Head esta seção contém informações sobre a página que não são visíveis para o usuário. É aqui que você define o título da página, que aparece na aba do navegador, e inclui links para arquivos CSS (estilo) e JavaScript (interatividade).

Tags

Tag ("P") Inicio

A tag P em HTML é usada para definir um parágrafo. É um dos elementos mais fundamentais para estruturar o conteúdo textual em uma página web. Os navegadores adicionam automaticamente um espaço (margem) antes e depois de cada elemento P para separar os parágrafos visualmente.

Tag ("input") Inicio

O elemento Input em HTML é usado para criar campos interativos em formulários, permitindo que os usuários insiram dados. Ele é extremamente versátil e seu comportamento varia de acordo com o valor do atributo type.

Principais Tipos de Input

  1. type="text": Campo de texto simples.
  2. type="password": Campo para senhas (oculta os caracteres digitados).
  3. type="email": Campo para e-mails (valida formato de e-mail).

Tag ("img") Inicio

A tag img em HTML é usada para incorporar imagens em uma página web. Ela é um elemento vazio, ou seja, não possui uma tag de fechamento, e requer atributos para funcionar corretamente

Principais atributos

  1. src (source): Define o caminho ou URL da imagem.
    • Exemplo: src="imagens/foto.jpg"
  2. alt (alternative text): Fornece uma descrição da imagem, usada para acessibilidade e como texto alternativo caso a imagem não carregue.
    • Exemplo: alt="Paisagem de montanha"
  3. width e height: Especificam a largura e altura da imagem (em pixels ou porcentagem).
    • Exemplo: width="300" height="200"
  4. title: Adiciona um texto que aparece como dica ao passar o mouse sobre a imagem
    • Exemplo: title="Descrição adicional"

Tag font Inicio

A tag font no HTML

A tag font era usada em versões antigas do HTML (como HTML 4) para alterar a aparência do texto, permitindo modificar cor, tamanho e tipo de fonte. No entanto, ela foi descontinuada no HTML5 e não é mais recomendada para uso, pois o HTML moderno utiliza CSS para estilização.

Iframe Inicio

O elemento Iframe em HTML é usado para incorporar outra página HTML dentro de uma página atual. Ele cria um "frame" embutido que pode exibir conteúdo de outra URL ou documento.
